## How To Start

### 1. Open Dashboard
```
http://localhost:5000
```

### 2. Add Sender Accounts
- Click "Sender Accounts (Add 5)"
- Click "+ Add Sender Account" 5 times
- Fill: Provider, Email, Password for each

### 3. Upload Leads
- CSV/Excel file with Business Name, Email, Location
- Click "Upload Leads"

### 4. Configure Campaign
- Campaign Name: (enter name)
- Emails Per Account: 30 (or custom)
- Add templates, subjects, follow-ups (optional)

### 5. Send
- Click "🚀 Send Campaign"
- Watch console and dashboard
- Each account sends to DIFFERENT leads
- Each email has 25-65 second random wait
- All 5 accounts sending SIMULTANEOUSLY!
