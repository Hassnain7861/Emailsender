## ⚡ HOW IT WORKS

### Parallel Sending (Not Sequential)

```
TRADITIONAL (Sequential):
Account 1 sends 30 emails → Account 2 sends 30 emails → Account 3 sends 30 emails...
Time: ~2-3 hours total

YOUR SYSTEM (Parallel - SIMULTANEOUS):
Account 1 sends 30 emails
Account 2 sends 30 emails  } ← ALL AT THE SAME TIME
Account 3 sends 30 emails  } ← NOT ONE AFTER ANOTHER
Account 4 sends 30 emails
Account 5 sends 30 emails

Time: ~1 hour total (5x faster!)
```

### Real Example: 150 Leads

```
Setup:
- 5 sender accounts (Gmail, Zoho, Outlook, Yahoo, SendGrid)
- 150 leads uploaded
- 30 emails per account

Execution:
┌──────────────────────────────────────────────────┐
│ ALL 5 ACCOUNTS START SENDING AT SAME TIME        │
├──────────────────────────────────────────────────┤
│ Account 1 → sends 30 emails to leads 1-30       │
│ Account 2 → sends 30 emails to leads 31-60      │
│ Account 3 → sends 30 emails to leads 61-90      │
│ Account 4 → sends 30 emails to leads 91-120     │
│ Account 5 → sends 30 emails to leads 121-150    │
└──────────────────────────────────────────────────┘

Total: 150 emails sent in parallel ✓
```

## ⚙️ TECHNICAL DETAILS

### Parallel Execution
- Each sender runs in separate thread
- Threads start with 0.5s delay (prevents thundering herd)
- All threads run simultaneously
- Main thread waits for all to complete

### Email Distribution
- Lead assignment:
  - Account 1: Leads 1-30
  - Account 2: Leads 31-60
  - Account 3: Leads 61-90
  - Account 4: Leads 91-120
  - Account 5: Leads 121-150

### Email Delays
- Between emails FROM SAME ACCOUNT: 3-8 seconds (random)
- Between different accounts: No delay (parallel)
- Prevents SMTP rate limits per account

### Templates & Subjects
- Templates rotate every 3 emails
- Subjects rotate per email (round-robin)
- Both distribute across all leads evenly

## 📞 SUPPORT

### Common Issues

**"Authentication Failed"**
- Check email address is correct
- Use APP PASSWORD (not regular password)
- Get new app password from provider

**"Cannot add 6th account"**
- System limited to 5 accounts max
- Remove one to add another

**"Campaign won't start"**
- Check all senders filled correctly
- Check leads uploaded
- Check at least 1 template exists

**"Emails sending slowly"**
- Intentional: 3-8s delays prevent spam
- Adjust if needed: Edit app.py, change `random.uniform(3, 8)`

**"Follow-ups not sending"**
- Keep Python window open
- Check scheduled time (wait for it to pass)
- Scheduler checks every 60 seconds
